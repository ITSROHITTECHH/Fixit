# Fixit
Fixit is a service platform for managing home improvement and maintenance tasks. It allows users to book skilled workers, explore service pricing, and request custom work. Designed with a clean user interface, responsive layout, and accessible UX. A step toward simplifying how we care for our spaces.     


# Fixit

Fixit is a modern home repair and maintenance service website that makes it easy for users to discover, compare, and book local service professionals. Designed with a clean UI and smooth browsing experience, Fixit turns everyday household problems into quick and reliable solutions.
